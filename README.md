# 🤖 Basic Chatbot

## 📌 Project Overview

This is a simple rule-based chatbot developed in Python as part of the **CodeAlpha Python Programming Internship**.

The chatbot responds to predefined user inputs such as greetings, questions, and exit commands.

## ✨ Features

- Greets the user
- Responds to predefined questions
- Handles unknown inputs
- Ends conversation using "bye"
- Console-based interaction

## 🛠️ Technologies Used

- Python 3

## 📚 Python Concepts Used

- Functions
- While Loop
- If-Elif-Else
- User Input
- String Methods

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/Mandeepsharma123/CodeAlpha_BasicChatbot.git
```

2. Open the project folder

3. Run

```bash
python basic_chatbot.py
```
